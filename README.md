# GameCast

## Development

Clone this repository:

```sh
git clone https://github.com/jTronixDevelopment/projectX.js
cd projectX
```

Install dependencies:

```sh
npm install
```

Start the project at [`http://localhost:3000`](http://localhost:3000).

```sh
npm start
```
